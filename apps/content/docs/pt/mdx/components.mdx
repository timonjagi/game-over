title: Componentes
description: Masterizando componentes em seus arquivos MDX
---

Os components são injetados automaticamente, sem a necessidade de importação manual, em todos os arquivos MDX.
Isso significa que você pode usar qualquer componente que esteja disponível!

Para saber mais sobre, adicionar ou alterar os componentes disponíveis, consulte o arquivo `src/components/docs/mdx.tsx`

## Alertas

**Exemplo:**

<Alert className="my-4">
  <AlertTitle>Alerta padrão</AlertTitle>

  <AlertDescription>
    Esse é um alerta padrão.
  </AlertDescription>
</Alert>

<Alert variant="warning">
  <AlertTitle>Alerta de aviso</AlertTitle>

  <AlertDescription>
    Esse é um alerta de aviso.
  </AlertDescription>
</Alert>

<Alert variant="destructive" className="my-4">
  <AlertTitle>Alerta de erro</AlertTitle>

  <AlertDescription>
    Esse é um alerta de erro.
  </AlertDescription>
</Alert>

**Código:**

```jsx
<Alert className="my-4">
  <AlertTitle>Alerta padrão</AlertTitle>

  <AlertDescription>
    Esse é um alerta padrão.
  </AlertDescription>
</Alert>

  <AlertTitle>Alerta de erro</AlertTitle>

    Esse é um alerta de erro.

```

## Guias

**Exemplo:**

<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Guia 1</TabsTrigger>
    <TabsTrigger value="tab-2">Guia 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Conteúdo da Guia 1
  </TabsContent>

  <TabsContent value="tab-2">
    Conteúdo da Guia 2
  </TabsContent>
</Tabs>

**Código:**

```jsx
<Tabs defaultValue="tab-1">
  <TabsList>
    <TabsTrigger value="tab-1">Guia 1</TabsTrigger>
    <TabsTrigger value="tab-2">Guia 2</TabsTrigger>
  </TabsList>

  <TabsContent value="tab-1">
    Conteúdo da Guia 1
  </TabsContent>

  <TabsContent value="tab-2">
    Conteúdo da Guia 2
  </TabsContent>
</Tabs>
```

## Etapas

**Exemplo:**

<Steps>
  <Step>Etapa 1:</Step>

  ```bash
  npx something
  ```

  <Step>Etapa 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>

**Código:**

````jsx
<Steps>
  <Step>Etapa 1:</Step>

  ```bash
  npx something
  ```

  <Step>Etapa 2:</Step>

  ```bash
  npx something-else
  ```
</Steps>
````
